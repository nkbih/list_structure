# list_structure

Позволяет просмотреть структуру файлов в нужной директории с заданным максимальным уровнем вложнности

```
>>> list_structure . 1
bsrc
  .vscode
  logs
  main.py
  utils.py
  venv
  __pycache__
fsrc
  .eslintrc.cjs
  .gitignore
  .vite
  index.html
  node_modules
  package-lock.json
  package.json
  public
  README.md
  src
  vite.config.js
  yarn.lock
```
